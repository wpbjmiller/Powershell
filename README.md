## Scripts

### Active Directory
- **[Export-ADUsers](./ActiveDirectory/Export-ADUsers/)** - Export Active Directory user accounts to CSV with comprehensive details

## Requirements
- Windows PowerShell 5.1 or later
- Appropriate modules for each script (see individual script documentation)
- Necessary permissions for the target systems

## Usage
Each script folder contains:
- The PowerShell script (`.ps1`)
- Individual README with detailed documentation
- Usage examples and requirements

Navigate to the specific script folder for detailed instructions.
